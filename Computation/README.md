# 3D Geometric Object Computation

## About The Project
This console application serves as the execution environment and tactile testing ground for the `GeometryLibrary`. It demonstrates the seamless instantiation, aggregation, and mathematical computation of 3D geometric objects (Tetrahedrons, Cuboids, Cylinders, and Composite Shapes). The architecture is designed to keep complex linear algebra and array management invisible to the end-user, providing a frictionless experience when executing geometry calculations.

### Built With
* .NET SDK (C#)

## Getting Started
To get a local copy up and running, follow these simple steps.

### Prerequisites
You must have the .NET SDK installed on your local machine to compile and run this application.
* Download .NET: [https://dotnet.microsoft.com/download](https://dotnet.microsoft.com/download)

### Installation
1. Extract the provided zip file (`Assignment2_MatriculationNumber_FirstName_LastName.zip`).
2. Open your preferred terminal.
3. Navigate to the root folder (`Computaion/`)
